# Privacy Policy

Last updated: 2026-03-29

NYT Connections Helper is an unofficial browser extension for customizing tile colors on the New York Times Connections game pages.

## What This Extension Does

- Runs only on these pages:
  - `https://www.nytimes.com/games/connections*`
  - `https://www.nytimes.com/crosswords/game/connections*`
- Lets you cycle tile colors and edit category names and colors.
- Stores only the extension's category settings in Chrome extension local storage on your device.

## What Data Is Stored

- Category count
- Category labels
- Category background colors
- Category text colors

## What Data Is Not Collected Or Sent

- No personal information is collected.
- No gameplay data is sent to the developer.
- No analytics, tracking, or advertising code is included.
- No extension data is transmitted to developer-controlled or third-party servers.
- No remote code or remotely hosted scripts are used by the extension.

## How Data Is Used

- Stored settings are used only to render your chosen category labels and colors inside the extension and on supported Connections pages.

## Data Sharing

- No data collected by the extension is sold or shared with third parties.

## Your Choices

- You can reset the stored settings from the popup or options page.
- You can remove all stored settings by uninstalling the extension.
